# Changelog

## 0.3.0 - 2026-02-10

- Added clearable UI support and `clear()` API.
- Added basic multi-select toggle behavior.
- Added custom render hooks (`renderOption`, `renderValue`).
- Added remote data loading (`load`) and static `data` option.
- Added minified builds and sourcemaps.

## 0.2.0 - 2026-02-10

- Added modular architecture under `src/` with a `FxdSelect` class.
- Added ARIA roles, keyboard navigation, and optgroup rendering.
- Added demo page and build/test scaffolding.
- Added `dist/` outputs for ESM, UMD, and CSS.
